# Evaluasi-Materi-Rasio-
Evaluasi Materi Rasio Kelas VII Berbasis Digital merupakan media penilaian yang dirancang untuk mengukur pemahaman peserta didik terhadap materi rasio. Media ini memanfaatkan teknologi digital untuk mendukung proses evaluasi yang lebih efektif, menarik, interaktif, dan mudah digunakan.
